## Install
- [ ]  Nodejs =v16
- [ ]  Npm >=8.19.3 or Yarn >=1.22.19

## How to start
1. Clone source code
   git clone https://github.com/playgroundvina/Friendify-GPT-frontend.git
2. Install dependencies
   ``` npm install ``` or ``` yarn```
3. ``` cp sample.env .env``` and change data .env file
4. Start code:
    + ```yarn dev``` or ```npm start``` to start source
